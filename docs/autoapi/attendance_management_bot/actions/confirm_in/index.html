attendance_management_bot.actions.confirm_in.deal_confirm_in(account_id, create_time, callback)¶

will be linked with the calendar internally, Check in time of registered user. Check also: attendance_management_bot/externals/calendar_req.py

Parameters

  • account_id – user account id.

  • create_time – current date by local time.

  • callback – The message content of the callback, include the user’s check-in time

Returns

Prompt message of successful check in.

attendance_management_bot.actions.confirm_in.confirm_in(account_id, current_date, create_time, callback)¶

This function is triggered when the user clicks confirm check-in. Update user’s input reminder status, progress.

Parameters

  • account_id – user account id.

  • current_date – current date by local time.

  • create_time – Time the request arrived at the server.

  • callback – User triggered callback.

Returns

None
